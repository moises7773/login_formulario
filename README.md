
# DevSpace Login UI

Una interfaz moderna de inicio de sesión y registro desarrollada únicamente con HTML, CSS y JavaScript puro.

El proyecto fue creado con fines de práctica para reforzar conceptos de desarrollo Frontend como manipulación del DOM, validación de formularios y almacenamiento local.

---

## Características

- Pantalla de bienvenida
- Inicio de sesión
- Registro de usuarios
- Validación de contraseña en tiempo real
- Diseño responsive
- Navegación entre pantallas sin recargar la página
- Persistencia de datos utilizando Local Storage

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

---

## Validaciones implementadas

La contraseña debe contener:

- Mínimo 8 caracteres
- Una letra mayúscula
- Un número
- Un carácter especial

La validación se realiza en tiempo real mientras el usuario escribe.

---

## Funcionamiento

El usuario puede crear una cuenta desde el formulario de registro.

Los datos se almacenan utilizando Local Storage del navegador.

Posteriormente el formulario de inicio de sesión compara el correo y la contraseña ingresados con la información almacenada para validar el acceso.

---

## Importante

Este proyecto fue desarrollado únicamente con fines educativos.

El uso de Local Storage para almacenar contraseñas **no es una práctica segura** y solamente se utilizó para practicar JavaScript del lado del cliente.

En una aplicación real la autenticación debe realizarse mediante un servidor y las contraseñas deben almacenarse cifradas utilizando algoritmos especializados.
